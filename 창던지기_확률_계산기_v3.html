창이 떨어진 존

등급별 등장 확률 — 클릭하면 개체 확률 펼침

1단계 — 등급 선택 확률
P(gi) = (1 − t) × P_base(gi) + t × P_luck(gi)
gi 등급 인덱스 — 0=Common · 1=Rare · 2=Epic · 3=Legendary · 4=Mythic · 5=Brainrot God · 6=Secret · 7=OG
k 창이 떨어진 존의 최고 등급 gi + 1  (예: Secret존 → k=7, OG존 → k=8)
t = L / 1000 = 0.00 행운력 혼합 비율 (0=기본분포 그대로, 1=행운분포로 완전 전환)
P_base W_base(gi) = 1 + r1×(k−1−gi) 로 정규화 — r1 클수록 하위 등급이 더 자주 등장
P_luck W_luck(gi) = (gi+1)^n 으로 정규화 — n=0이면 균등, n 클수록 최상위에 집중
r1 = 0.50 감쇠계수 (등급용)
L = 0% 행운력 — 클수록 P_luck 비중 증가
n = 1.0 집중력 — 클수록 행운 효과가 최상위 등급에만 집중
2단계 — 등급 내 개체 선택 확률
W(j) = 1 + r2 × rank[j]
j 등급 내 개별 브레인롯
rank[j] 수익 내림차순 순위 (0 = 최고수익, 동수는 같은 rank)
r2 = 0.50 감쇠계수 (개체용) — 클수록 저수익 개체가 더 자주 등장
