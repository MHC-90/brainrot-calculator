브레인롯 뽑기 — 등급 선택 확률
등급확률(gi) = (1 − t) × 기본분포(gi) + t × 행운분포(gi)
기본분포(gi) 가중치 = 1 + r1×(k−1−gi) — 하위 등급일수록 가중치 높음
행운분포(gi) 가중치 = (gi+1)^n — 상위 등급일수록 가중치 높음
t = L / 1000 = 0.00 행운력 혼합 비율 (0=기본분포, 1=행운분포)
gi 등급 인덱스 — 0=Common · 1=Rare · 2=Epic · 3=Legendary · 4=Mythic · 5=Brainrot God · 6=Secret · 7=OG
k 존의 최고 등급 gi+1 (Secret존=7, OG존=8)
r1 = 0.50 감쇠계수 (등급용)
L = 0% 행운력
n = 1.0 집중력
브레인롯 뽑기 — 등급 내 개체 선택 확률
가중치(j) = 1 + r2 × rank[j]
개체확률(j) = 가중치(j) / 전체 가중치 합
rank[j] 수익 내림차순 순위 (0=최고수익, 동수는 같은 rank)
r2 = 0.50 감쇠계수 (개체용) — 클수록 저수익 개체가 더 자주 등장
별로직 1단계 — 발동 확률
발동 확률 = 1 − perm(M, N) / M^N
perm(M, N) M × (M−1) × ... × (M−N+1) — 창 N개가 모두 다른 브레인롯일 경우의 수
M^N 전체 경우의 수
현재 발동 확률 = —% N=—개, M=—종
별로직 2단계 — 브레인롯 등급 선택
브레인롯 뽑기 등급 선택 확률과 동일
주의 3단계에서 독립시행 결과에 없는 브레인롯만 후보로 사용
별로직 3단계 — 등급 내 개체 선택
브레인롯 뽑기 개체 선택 확률과 동일
주의 독립시행에서 이미 뽑힌 브레인롯은 후보에서 제외
별로직 4단계 — 별 갯수 정하기 (룰렛)
가중치(k) = base^(k−2) × k^(n×t)
base = (N/M)^(0.5×(1−t))
k 별 갯수 (2~N) — 가중치가 높을수록 더 자주 뽑힘
t = L / 4000 = 0.0000 행운력 조절값 (L=0→t=0, L=1000%→t=0.25)
n = 1.0 집중력 — 클수록 행운력 효과가 고별에 집중
